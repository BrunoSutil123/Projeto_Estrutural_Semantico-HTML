# Fundamentos-Avan-ados-de-HTML-Projeto-Estrutural-Sem-ntico
Projeto de Landing Page desenvolvida exclusivamente com HTML, focada em estrutura semântica, hierarquia correta de headings, SEO e acessibilidade. Aplica boas práticas como uso adequado de tags estruturais, meta description estratégica e atributos alt em imagens, demonstrando domínio da base essencial do desenvolvimento Front-end.
